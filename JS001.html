JS001 - A JavaScript elhelyezése

kulcsszavak

[document.write | Js elhelyezése]

Téma

Hol helyezhetjük ela JavaScript kódokat ?

  1. A weboldal törzsében script tagpár közé zárva. Az oldal feltöltésekor fut le a Js és az eredmény az oldal HTML kódjába kerül.

Példa-1.:
<p>
<scrip language="JavaScript" type="text/javascript">
document.write(document.lastModified)
</script>
</p>
